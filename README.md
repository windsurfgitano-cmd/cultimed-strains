# CULTIMED - Premium Cannabis Strains

Proyecto Next.js con 6 landing pages de cepas premium.

## Cepas Incluidas
1. Dante's Inferno - MAD TIKI
2. Gaslight - LIT FARM
3. Bourbon Street - LIT FARM
4. Zoap Mintz - LIT FARM
5. Cherry Zoda - UMAMI SEED CO
6. Watermelon Z 2.0 - KNOWN WASHERS

## Rutas
- `/` - Home con todas las cepas
- `/dantes-inferno` - Dante's Inferno
- `/gaslight` - Gaslight
- `/bourbon-street` - Bourbon Street
- `/zoap-mintz` - Zoap Mintz
- `/cherry-zoda` - Cherry Zoda
- `/watermelon-z-2` - Watermelon Z 2.0

## Deploy
```bash
npm install
npm run dev
```

Luego deployar en Vercel para obtener URLs individuales por cepa.
